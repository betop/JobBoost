Options

Configuration
This extension reads backend API settings from config.json at the extension root. Edit that file and then reload the extension in chrome://extensions.
AI Backend
Set backendApiUrl to your Xano endpoint (for example /public/gmail-analyze). The backend calls Claude Haiku 4.5 using your server-side Anthropic key.
Google OAuth
This extension uses chrome.identity. You must set your Google OAuth Client ID in manifest.json (oauth2.client_id) and keep the scope https://www.googleapis.com/auth/gmail.modify enabled.
