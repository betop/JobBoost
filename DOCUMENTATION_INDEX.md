## 📞 Support & Help

### If you're stuck:
1. Check the relevant README for your component
2. Review SETUP_GUIDE.md troubleshooting section
3. Check QUICK_REFERENCE.md for common issues
4. Review browser/terminal console for errors
5. Verify all configuration files are correct

### Common Issues & Solutions:
- **Admin panel won't start** → Check Node.js version, npm install
- **Extension won't load** → Check manifest.json, verify file paths
- **API calls failing** → Check API URL in config, verify backend running
- **Token validation failed** → Check token.json, verify token is valid
- **PDFs not generating** → Check jsPDF loading, browser console
